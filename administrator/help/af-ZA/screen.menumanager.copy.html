Menu Bestuurder: Kopieer Menu

-- Voorlopige Weergawe

Hierdie bladsy let's jy copy 'n Menu en all sy Menu Items.
'n lys vertoon die naam of die Menu en Menu Items copied.

Nuwe Menu Name:
Type hier to create 'n nuwe Menu Name.

Hierdie is die identifikasie naam used by Joomla to identifiseer hierdie menu metin the code ; it moet be unieke. Dit is recommended to gebruik no space metin hierdie name. For voorbeeld, in die core Joomla! die Gebruiker Menu's Menu Name is gebruikermenu en die Main Menu's Menu Name is mainmenu.

New Module Name:
Type hier 'n Module Titel.

Hierdie is die naam given na die new mod_mainmenu module which is outomaties created when jy stoor hierdie menu. Dit is ook die Name of die menu soos dit sal verskyn on the bladsy if jy Kies to show it (this may be changed when redigeering die module). In die voorbeeld beskryf above these is Gebruiker Menu en Hoof Menu.

Taakbalk
Kopieer:
'n Nuwe Menu Name sal be created in die Menu Bestuurder lys where it may be geredigeer.
'n Nuwe mod_mainmenu module, met die Name jy het entered above, sal ook be created.
Parameters vir die module created is to be geredigeer deur die 'Modules Bestuurder [site]': Modules -> Werf Modules.
