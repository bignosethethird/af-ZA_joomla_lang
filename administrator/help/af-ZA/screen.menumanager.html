Menu Bestuurder

"Menus" in Joomla is groepe of navigational links to seksies, kategorië, inhoud items, komponente of external bladsys. Hierdie links is called 'Menu Items'.

Elke menu moet het 'n identifikasie naam which is only used internally by Joomla.
'n menu only becomes visible on 'n site if it has 'n publiseered mod_mainmenu module which references it. 'n menu may het multiple mod_mainmmenu modules referencing it.

Taakbalk Ikone:
New - Kliek hierdie ikoon to create 'n nuwe Menu. You sal access die "Menu Details" venster. Type 'n Menu Name. Hierdie is die identifikasie naam used by Joomla to identifiseer hierdie menu met in die code. It moet be unieke. Dit is recommended to gebruik no space metin hierdie name. For voorbeeld, in die core Joomla die Gebruiker Menu's Menu Name is 'gebruikermenu' en die Hoof Menu's Menu Name is 'mainmenu'.

Next, type die Module Titel. Hierdie is die naam given na die new mod_mainmenu module which is outomaties created when jy stoor hierdie menu. Dit is ook die Name of the menu soos dit sal verskyn op die bladsy if jy Kies to publiseer it. Hierdie may be changed in die future when redigeering die module. In die voorbeeld beskryf above, these is 'Gebruiker Menu' en 'Hoof Menu'. Kliek die 'Stoor' ikoon. Your new menu verskyns in die Menu Bestuurder lys.

Redigeer - Kliek die radio knoppie langs die Menu Name jy want to redigeer. Dan, kliek die "Redigeer" ikoon which sal take jy na die Redigeering bladsy.

Kopieer - Kliek die radio knoppie langs die Menu Name jy want to copy. Dan, click die 'Kopieer' ikoon in die taakbalk. Hierdie sal duplicate die menu. Jy mag copy any van jou menus. Kopieering sal create 'n nuwe Menu Name, containing dieselfde Menu Items as die menu copied, associated to 'n nuwe Menu Module.

'n 'Kopieer Page' sal vertoon, metdieselfde choices as in Create Nuwe Menu . It sal ook status die naam of die Menu being copied en 'n lys van die Menu Items associated na die menu being copied. Fill in die fields as in die create "New" Menu screen.

Skrap - Kliek die radio knoppie langs die Menu Name jy want to delete. Dan, kliek die 'Skrap' ikoon in die taakbalk. Hierdie sal skrap die Menu. 'n "Skrap Menu : Name-of-the-menu" skerm sal vertoon. It vertoon die Module(s) to be deleted (the mod_mainmenu modules associated to hierdie Menu) en die Menu Items to be deleted. Kliek die Skrap Icon surrounded by 'n dotted line. Hierdie sal delete hierdie Menu, ALL sy Menu Items en ALL die Modules associated metit. 'n Javascript alert popup boksie gives jy die choice to "OK" of to "Kanselleer" hierdie aksie.
LET OP: Skrapd Menus kan nie be recovered van die Vuillis Bestuurder!


Kolomme:
Die headings give inligting op die volgende funksies :

Menu Name - Kliek op die naam to redigeer it. Hierdie name may be die one that is given na die menu by verstek of die naam jy gave to that menu.

Menu Items - Kliek die 'Redigeer' ikoon in hierdie kolom to add of redigeer Menu Items associated meta menu. Or alternatively gaan na "Menu->the-menu-jy-choose-to-edit". 'n "Menu Bestuurder [the-menu-jy-choose-to-edit]" venster sal vertoon met 'n lys of all Menu Items associated to hierdie Menu.

#Gepubliseer - Hierdie is die aantal Menu Items publiseered.

#Ontpubliseer - Hierdie is die die aantal Menu Items ontpubliseered.

#Vuillis - Hierdie is die aantal Menu Items in die Vuillis Bestuurder.

#Modules - Hierdie is die aantal menu modules associated na die Menu Name.
