Menu Bestuurder / Hoof Menu en other Menus

Hierdie bladsy vertoon 'n lys van die Menu Items associated na die menu jy het just chosen.

Taakbalk Ikone:
Nuut : Kliek hierdie ikoon to create 'n nuwe Menu Item
Redigeer : Kliek hierdie ikoon 'n bestaande Menuitem te redigeer.

Publiseer en Ontpubliseer : Hierdie ikone het dieselfde funksie as die 'Gepubliseer' kolom ikoon. Kliek òf of these ikone to publiseer of ontpubliseer any items. Jy mag gebruik these ikone metmultiple items checked.

Move : Kliek hierdie ikoon to open 'n nuwe 'Move Menu Items' venster waar jy kan Kies to move die spesifieke Menu Item(s) 'and children if desired' to ander bestaande Menu. Jy mag check multiple items to be moved at once.

Kopieer : Kliek hierdie ikoon to open 'n nuwe 'Kopieer Menu Items' venster waar jy kan Kies to copy die spesifieke Menu Item(s) ''and children if desired'' to ander bestaande Menu. Jy mag check multiple items to be copied at once.

Vuillis : Kliek hierdie ikoon to send die Menu Item na die Vuillis Bestuurder. Jy mag check multiple items to be sent na die Vuillis Bestuurder at once. Die menu items may be recovered deur die Admin Menu: Werf->Vuillis Bestuurder. Dan, choose die Menu Items tab.

Kanselleer : Kliek hierdie ikoon to cancel any wysig made in die 'Order' kolom if die 'Stoor Order' ikoon has not been clicked. You sal return na die previous bladsy.

Kolomme:
Menu Item : Hierdie is die naam van die skakel soos dit op jou werf verskyn. Kliek hierdie naam so dat jy kan redigeer parameters related to hierdie Item. 'n slot ikoon langs it vertoon that it is checked-out. To check-in, kliek die name if it is jy that has last geredigeer die bladsy. Dan kliek die 'Stoor' of 'Kanselleer' icon to check back in die item. Indien 'n ander gebruiker has checked out die item, please kontak him/her of die administrateur of jou werf.

Gepubliseer: Hierdie vertoon whether die Menu Item is publiseered of ontpubliseered. Kliek die ikoon to switch sy status.

Reorder : Hierdie vertoon die order of Menu Items in die menu. Kliek die arrow icons to move die item up of down in die lys.

Order : Hierdie vertoon die order of Menu Items in die menu. To wysig die order, type in die order number jy wish to het. Jy kan wysig die order of any two items of meer hierdie way. Dan kliek die 'Stoor Order' ikoon langs the Kolom Heading 'Order'.

LET OP: Die Order en Reorder kolomme show die relative posisies of Parent en Children Menu Items. Moving 'n parent moves die children too. Die order numbers is vertoon relatively na die Parent Item.

Toegang : Kliek die 'Toegang' level to wissel to ander Gebruiker toegangsvlak vir this menu item.

Item Id: Hierdie vertoon die 'id' vir hierdie spesifieke Menu Item in die databasis.

Tipe: Hierdie is die type of skakel used vir hierdie Menu Item. Indien it is 'n 'Link '“ Inhoud Item' of 'n 'Link '“ Statiese Inhoud', jy may directly redigeer die inhoud by kliek die type of menu item.

CID: Hierdie vertoon die 'componentid' vir hierdie spesifieke Menu Item in die databasis.

Other funksies beskikbaar above en below of die lys:

Max Levels: Kies die maximum aantal levels deep that die menu tree is vertoonde on hierdie admin bladsy.

Filter: Type in die sleutelwoord to search vir teks in die Menu Item titels.

Vertoon #: Kies die aantal menu items jy wish to be vertoonde in die lys.
LET OP: Hierdie setting sal be preserved vir all lys on all screens while jy are in die Administator back-end. 'n verstek value vir die lys length is ook configurable in Globale Konfigurasie.
