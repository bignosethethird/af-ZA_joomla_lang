Wat is Nuut in Weergawe 1.0

Die volgende beskryf die hoogtepunte van die debut weergawe van Joomla! Alle kommentaar, veranderinge en verbeteringe is met verwysing tot Mambo 4.5.2.3 gemaak .

Fout Verbeteringe

Several crucial fixes het been applied to hierdie weergawe:

  • Jy kan now properly login to different sites even if they share dieselfde domain en is nested in sub-directories.
  • Slow queries in die Gebruiker Bestuurder het been corrected. Hierdie bug affected sites metthousands of registered gebruikers. Registered gebruikers (that don't het redigeer rights) is ook excluded van die Outeur select lys vertoon when redigeering inhoud bladsys.
  • Variable references that caused notices in PHP 4.4 het been fixed.

Check die CHANGELOG.php lêer vir meer inligting.

Security Fixes

Several important vulnerabilities het been patched in hierdie weergawe:

  • Known vulnerabilities in die phpMailer class
  • SQL injection vulnerability via die gebruiker activation feature
  • SQL injection vulnerability via die peilings component
  • Lêërs such as die CHANGELOG het been converted to PHP l lêersecirc; lêerseuml;rs to prevent them being visible.
  • The "E-pos van Friend" form has been hardened to reduce die risk of it being used as 'n gateway vir spam
  • The mosGetParam has been hardened by using phpInputFilter vir NO_HTML mode
  • 'n alternative weergawe of globals.php has been provided. Indien jy renaam globals.php-off to globals.php Joomla! sal emulate die php setting register_globals=off. Hierdie affords some protection against certainly types of malicious attacks maar may prevent some third party toepassings van working correctly.

Joomla! Werf (Front End)

  • die site is now visibile to Administrateurs even when die site is in Offline mode.
  • Frontend Gebruikers kan select die WYSIWYG redigeerder they want to gebruik rather than having to gebruik die verstek.

Joomla! Administrateur (Agterkant)

  • The Stelsel Information skakel has been added terug na die Stelsel Menu.
  • "Force Logout" in die Gebruiker Bestuurder taakbalk has been renamed to simply "Logout".
  • "Submit - Inhoud" is 'n nuwe menu type beskikbaar
  • HTML output is now buffered to improve performance of non-vertoon aksies (like saving inhoud)
  • Gebruikers kan select die WYSIWYG redigeerder they want to gebruik rather than having to use die verstek.
  • Kontak component: added parameter to defineer 'n certain kategorie vir 'n kontak menu.

Templette

Die Solar Flare 2 templet has been freshened met die new name. Andy Miller joined forces metRobert Deutz to deliver 'n stunning new Adminstrator Templet vir Joomla!

Taakbalk ikone in die Administrateur taakbalk is now vertoon in full colour rather than greyed out (which would normally indicate they were unbeskikbaar).

Miscellaneous Bits

  • 'n "check again" knoppie has been added na die eerste bladsy of die web installeerder.

Ondersteuning vir MySQL 4.1.x

Joomla! sal ondersteun installation en gebruik metPHP 5 en MySQL 4.1. 'n Alternative databasis lêer is provided in die /includes lêergids. SImply copy databasis.mysqli.php over die bestaande databased.php lêer. Following this jy kan gebruik die web installeerder normally to install Joomla!.

Library Opgradeerings

Die volgende ontwikkellaar libraries is nou opgradeer:

  • patTemplate (this weergawe now supports die kort tag modifiers)
  • Overlib
  • TinyMCE
  • Walter Zorn tooltips
  • JSCookMenu
  • DOMIT!
  • DOMIT! RSS

Gebruikbaarheid en Toeganglikheid

'n huge thrust is being made to make Joomla! compliant metweb standards vir useability en accessibility. While much meer sal be done in future versions, die volgende eerste steps het been taken to achieve hierdie goal:

  • Label tags het been added to Werf module en component output
  • mosHTML::selectList en mosHTML::radioList outomaties provide die correct label tags

Ontwikkellaar Ekstras

'n Aantal nuwe dinge is beskikbaar vir ontwikkellaars. Hulle is:

  • patHTML class provides several utility methods vir handling selection lys
  • mosAbstractTasker class now auto-maps public methods to tasks
  • setTemplateVars method has been added to both die Werf en Administrateur bladsy navigation classes
  • mosHash funksie provides 'n convenient method vir producing secure hashes
  • 'n <span class="article_separator"></span> now separates inhoud items in lys
  • mosParameter::_mos_filelys has been added to supportXML params
  • Paramater l lêersecirc; lêerseuml;rs kan now gebruik die root element <mosparams>. Hierdie enables jy to het meer than one XML params lêer in jou component installation l lêersecirc; lêerseuml;rs.
  • 'n aantal new mambot triggers het been included: onStart, onAfterMainframe
  • 'n aantal new taakbalk ikone het been added
  • The mosToolBar::help method now correctly supports local hulp l lêersecirc; lêerseuml;rs in 'n komponente /help folder.
  • mosDBTable::isCheckedOut method has been added
  • CSS styles het been added na die administrateur templet to ondersteun Joomla! 1.5-like (Mambo 4.5.3-like) skerm layouts.
  • The dTree javascript tree library has been added
  • databasis:setQuery now takes arguments vir query limit offset en aantal records to be returned.


Versoenbaarheid

We het made every pooging to make hierdie weergawe terugwaards versoenbaar met Mambo 4.5.2.3 maar daar is 'n few changes that may affect 'n small selection of bestaande addons. Indien jy is 'n ontwikkellaar of Mambo addons (and soon to be Joomla! ontwikkellaar) then jou attention should be drawn na die volgende points:

  • patTemplate::createTemplate arguments het changed
  • databasis:setQuery arguments het changed maar hierdie is unlikely to affect most sites
