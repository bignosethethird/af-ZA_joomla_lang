Stelsel Informasie

-- Voorlopige Weergawe

Die inligting is vertoon in 'n tabbed formaat met drie hoof inhouds-afdelings.

Stelsel Tab:
Hierdie 'n broad snapshot van die server en versions of die current installation komponente van jou Joomla site. Hier jy sal find jou PHP build en weergawe, jou MySQL databasis weergawe along met die Joomla weergawe en relevante settings that jou server has set that might interfere metyour current site installation.

Konfigurasie:
Hierdie vertoon die inhoud van jou Joomla configuration.php-lêer. Hierdie lêer word produced when jy perform die installation of Joomla op jou server, en is changed when jy make changes to in die global configuration seksie of die Administrasie of Joomla. Hierdie lêer defineer many of die global variables that Joomla uses to execute. Jy kan not redigeer die lêer van hier, maar it does allow jy to look at jou current settings to find problems of to copy die inligting vir 'n ondersteun post op die forum on forum.joomla.org.

PHP:
PHP is die server side scripting taal that Joomla runs on. Hierdie taal is installeer en executes op die server, en therefore all die settings is made on that server. Everything jy ever wanted to know about die PHP installation op jou Joomla server is lysed hier. How much control jy het over changing hierdie inligting depends on whether jy own die server of if jou server host is willing to work metyou on needed changes. Die PHP tab sal allow jy to look at all die settings en variables that defineer jou PHP installation. Dit is 'n good practice to insure that jy know what requirements is needed vir any 'Derde Party' Komponente that jy plan to gebruik metJoomla. Jy kan then check vir those requirements hier.
