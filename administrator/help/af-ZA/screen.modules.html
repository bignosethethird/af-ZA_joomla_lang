Module Bestuurder

Modules is modular elements used to vertoon inligting items of interactive features throughout 'n Joomla Werf. Die main characteristic is to be posisieed off inhoud in pre-established posisies ruled by die corresponding templet (Front-end of Back-end templet vir Werf en Administrateur areas respectively).

Voorbeelde of modules is Menus, Login en Soektogvorms. Eenvoudige HTML Modules kan be created directly van die Module Bestuurder. In meer complicated cases, they sal het to be installeer. There is hundreds of modules reeds programmed vir Joomla, written by various Joomla community members. You sal find them op die Joomla projects site, http://ontwikkellaar.joomla.org. Hulle is gratis en beskikbaar vir download.

Die Module Bestuurder vertoon die installeer modules. Jy kan manage them by kliek op die appropriate ikoon in die Tool Bar.

Taakbalk Ikone:
Publiseer: Kliek die 'Publiseer' ikoon to publiseer jou modules op die Front-end van jou webwerf.

Ontpubliseer: Kliek die 'Ontpubliseer' ikoon to ontpubliseer jou modules op die Front-end van jou webwerf.

Kopieer: Kliek die 'Kopieer' ikoon to make 'n copy of die selected module. Hierdie enables it to be vertoonde in meer than one place op die Front-end van jou webwerf. 'n Nuwe weergawe is created met die naam 'Kopieer of (the name of die module jy copied)'. Jy kan renaam it.

Nuut: Kliek die "New" ikoon to create 'n custom module meta "Gebruiker" type. Hierdie is 'n very useful module which may be used to vertoon Inhoud (this inhoud is not related to Seksies, Kategorië of Statiese Inhoud Items) of 'n RSS Feed. It kan be used as 'n teks boksie, image module of even to include customized code.

Redigeer: Kliek die 'Redigeer' ikoon to redigeer of modify die module. Klieking die linked name sal ook perform hierdie funksie. Hierdie enables jy to wysig die attributes of 'n module like it's titel, it's posisie, die order in which it verskyns op die bladsy, toegangsvlak en whether it is publiseered of not. From hierdie bladsy jy sal ook be able to wysig die location die module verskyns op die Front-end of jou werf and any parameters that apply na die module.

Skrap: Kliek die 'Skrap' ikoon 'n module van jou system te skrap/ontinstalleer.
LET OP: Jy kan nie skrap die 'last' module of 'n particular 'type' on hierdie screen. Indien jy want to skrap 'n spesifieke type of module, go na die Install/Uninstall skerm as explained above.

Hulp: Kliek die 'Hulp' ikoon om die hulp skerm vir hierdie bladsy te besigtig.


Kolom Beskrywings:

Check Box/Padlock: Kliek hierdie to select the module. 'n slot ikoon sal vervang die Check Box if die item is checked-out. Hierdie means die module is unbeskikbaar to other managers. If jy het checked-out die item, kliek die naam to go na die Redigeer screen. Jy kan òf kliek the Stoor òf die Kanselleer ikoon to check back in die module. If hierdie module has been checked out by ander manager/administrateur, please kontak die administrateur of jou werf vir him/her to Check-in die module.

Module Naam: Hierdie is die module se titel. Dit is redigeerbaar and sal be vertoonde op die Front-end van jou webwerf if jy Kies so.

Gepubliseer: Hierdie dui aan jy if die module inhouds sal show op die Front-end van jou webwerf. Kliek die 'blue' arrow ikoon of 'red' cross to wissel die status.

Reorder: Hierdie kolom allows jy to wysig the relevante order that die modules vertoon op die Front-end van jou Web bladsy(s). Kliek die 'up' of 'down' arrows to increase of decrease die module order by one posisie.

Order : Hierdie vertoon die order of die modules as vertoonde in die Front-end van jou site.
To modify hierdie order, type in die number that jy would like to het them presented. Dan, kliek die 'Stoor Order' ikoon langs die Kolom Heading 'Order'.

Toegang: Hierdie vertoon wie kan see die modules op die Front-end of 'n Web bladsy. Klieking die skakel sal wissel deur Public/Registered/Special.

Posisie: Hierdie vertoon which templet posisies die module sal vertoon in. Party common Templet posisies is - Gebruiker1, Left, Right, en Top.
LET OP: Jy kan only assign die module posisie to one that is reeds coded into jou templet

Pages: Hierdie vertoon which bladsys die module sal vertoon on. Opsies include All, Varies of None.
LET OP: Hierdie is not applicable to Administrateur modules. Kliek op die Redigeer ikoon in die Taakbalk om hierdie te redigeer.

ID: Hierdie is die ID number vir die module item. Hierdie is allocated by die Joomla site system when die module instance is saved.

Type: Hierdie is die type of module. 'n voorbeeld is mod_mainmenu which is die 'type' of module used vir die Gebruiker Menu en die Hoof Menu module instances.

At die bottom of die lys jy may see 'n pagination feature, which sal allow jy to navigate na die next set of modules in die lys. Jy kan ook set die number of modules vertoonde on 'n bladsy by selecting die number in die aftrek boksie at the bottom of die screen.

Filtering funksies is beskikbaar at die regs-bo aan die lys. To filter die vertoonde modules volgend Posisie en module tipe, kies van die Kies Posisie en/of Kies Tipe aftrek menu. Jy kan refine jou filter by typing in die sleutelwoord that jy wish to filter in die field boksie.
