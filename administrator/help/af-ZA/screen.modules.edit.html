Modules: Redigeer

Hierdie bladsy vertoon die redigeerbaar parameters for Modules in Joomla.
See bottom of bladsy vir spesifieke module help.

Taakbalk Ikone:
Voorskou: Kliek die "Voorskou" ikoon to preview only typed modules.

Stoor : Kliek die "Stoor" ikoon to stoor die parameters jy het chosen and to go terug na die 'Menu Bestuurder'.

Toepas : Kliek die "Toepas" icon to stoor die parameters jy het chosen en to keep dieselfde venster open vir redigeering.

Sluit : Kliek die "Sluit" icon to cancel die aksie en to sluit die "Modules: Redigeer" venster.

Details:
(dieselfde vir alle modules)

Titel: Type of Redigeer die titel of die module that jy wish to vertoon on the Front-end of Back-end of jou werf.

Show titel: Kies if jy wish vir die "Titel" to vertoon of not on the Front-end of Back-end of jou werf.

Posisie: Kies which posisie, op die Front-end of Back-end of jou werf, the module sal vertoon in.

Module order: Kies die order die module sal verskyn among modules assigned to die Front-end of Back-end of jou werf.

Toegang Level: Kies wie kan view hierdie module. Public (everyone kan view it), Registered (only registered gebruikers kan view it), of Special (only special gebruikers kan view it, as defined under die Gebruiker Bestuurder in Werf -> Gebruiker Bestuurder -> Add/Redigeer Gebruikers).

Gepubliseer: Kies "Ja" to publiseer die module op die Frontend van jou webwerf (if chosen Posisie is beskikbaar in die templet) of "Nee" to ontpubliseer die module.

ID: Hierdie is die ID number vir die module item. Hierdie is toegewys duer die Joomla werf stelsel wanneer die module gestoor word.

Beskrywing: Hierdie is 'n kort beskrywing van die funksies wat die Module aanbied.

Pages / Items (same vir any module): Choose which bladsys jy want die module to be vertoonde on.

Parameters:
Ontsper Cache (beskikbaar vir many modules): Kies to cache of not die inhoud of hierdie module.
Other parameters may wysig depending on the module (see below).

Module Class Suffix (beskikbaar vir all module): Type (if needed) die suffix to be aplied na die CSS class of die module (table.moduletable) to allow individual module styling.
