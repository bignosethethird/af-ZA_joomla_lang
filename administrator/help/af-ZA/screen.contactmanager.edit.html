Kontak Bestuurder : Nuut/Redigeer

--- Voorlopige Weergawe

Hierdie bladsy laat jou create of redigeer kontakte op jou werf.


Taakbalk:
Stoor: Kliek hierdie ikoon to 'Stoor' jou new of geredigeer Kontak Details.

Kanselleer : Kliek hierdie ikoon to exit die skerm sonder saving. Hierdie takes jy back to Kontak Bestuurder. Die kontak remains ingeteken.

Kontak Details:
Enige veld left blank is not vertoonde.
Enige veld filled out sal be included inna die downloadable vCard vir hierdie Kontak.

Kategorie: Drop-down select of all publiseered Kontak kategorië

Linked to Gebruiker: Drop-down select of all registered gebruikers of die web site. Can be set to " - No Gebruiker - " vir independent setup of 'n site kontak.

Name: Type hier die naam of die kontak soos dit sal vertoon op die site. Can be òf 'n person's name, of 'n departement, 'n product, of whatever jy wish.

Kontak's Posisie: Best used meta Name, as die titel is vertoonde if hierdie field word gebruik. Example would be "CEO".

E-mail: E-mail adress vir hierdie Kontak.

Die volgende 5 fields kan be used to provide geographical access inligting if desired.
  • Street Address
  • Town/Suburb
  • State/County
  • Country
  • Postal Code/ZIP

Die volgende two fields kan be used to provide Phone en Fax Kontak inligting if desired.
  • Telephone
  • Fax
Miscellaneous Info: Type hier any other inligting jy wish to provide about hierdie Kontak.


Publiseering Tab:
Werf Verstek: Kies Yes of No to make hierdie Kontak die verstek vir jou werf.

Gepubliseer: Kies Yes of No to show hierdie Kontak to jou besoekers.

Ordering: Kontak needs to be saved eerste to vertoon 'n aftreklys select hier to order hierdie Kontak in die lys of publiseered Kontakte vir die site. Kies die Kontak after which jy wish to vertoon die current Kontak.

Toegang: Wie kan view hierdie Kontak op die frontend if publiseered. Public, Registered of Special

Images Tab:
Jy mag select hier 'n prent (van folder images/stories) to vertoon langs this Kontak, of select "- Kies Image -" to not vertoon any image.

Indien 'n prent is chosen, it sal show in die Voorskou block.

ParametersTab:
Hier is various settings vir what is vertoonde when die gebruiker clicks to view 'n Kontak item.

Menu Image: Allows jy to set 'n prent that would verskyn na die left of right of die menu item. Die image moet be placed in die /images/stories lêergids.
Note: Hierdie field is useless in 4.5.2.1 as hierdie bladsy does not concern 'n Menu Item.

Page Class Suffix: Allows jy to set customized styles vir hierdie bladsy. Die suffix jy type hier is outomaties added na die CSS classes vir hierdie bladsy.

Afdruk Ikoon: Hides of vertoon die ikoon en skakel na die printer-friendly format of items. Die pull down laat select 'Use Globale' to gebruik die parameter as defined globally deur die Werf->Globale Konfigurasie, 'Hide' of 'Show'.

Back Knoppie: Beheers whether to show of hide 'n "Back" teks skakel that would verskyn op die bottom of die bladsy en return die gebruiker na die previously viewed bladsy. Die pull down laat select 'Use Globale' to gebruik die parameter as defined globally deur die Werf->Globale Konfigurasie, 'Hide' of 'Show'.

Die volgende 12 fields defined above may be hidden of vertoon by checking die radio knoppies:
  • Name
  • Posisie
  • E-mail
  • Street Address
  • Town/Suburb
  • State/County
  • Country
  • Postal Code/ZIP
  • Telephone
  • Fax
  • Misc. Info
  • Image
E-mail beskrywing: Show/Hide die beskrywing teks below.

Beskrywing text: Type hier die beskrywing teks vir die E-pos form. Indien left blank, it sal gebruik die _EMAIL_DESCRIPTION taal definition.

E-mail Form: Show/Hide die e-pos to form.

E-mail copy: Show/Hide die merkboksie to e-pos 'n copy na die sender's adress.

Drop-down: Hide of Show die aftreklys select lys when in full view. Whatever the kontak skakel accessed, hierdie sal let die gebruiker Kies ander kontak to vertoon.

Ikone/Text: Hierdie laat jou decide if die fields lysed below should be introduced by 'n icon, die verstek teks vir die fields in die Kontak Details of nothing.

Jy mag kies, if decided so above, 'n spesifieke ikoon to introduce die volgende velde:
  • Address
  • E-mail
  • Telephone
  • Fax
  • Misc Info
Indien jy Kies 'n custom image, hierdie image should be uploaded na die images/stories lêergids in die Media Bestuurder
