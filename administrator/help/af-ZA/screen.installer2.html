There is two ways to install one of these elements.

Oplaai Pakket Lêer:
Providing jy het GZip ondersteun metPHP, jy sal be able to upload 'n zipped paket lêer that sal be outomaties installeer to jou server. 'Browse' na die location of die element op jou rekenaar. Dan kliek die 'Oplaai Lêer & Install' ikoon.

To install van lêergids:
Indien jy don't het GZip ondersteun then jy may upload 'n unpacked argief to jou host's server. Type die location of hierdie lêer (it moet be die absolute location) then click die 'Install' ikoon.

To uninstall one of these elements:
Tick die boksie langs die Element (or multiple elements) jy want to uninstall, then kliek die 'Skrap' top menu ikoon.
