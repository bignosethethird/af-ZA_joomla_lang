Menu Bestuurder : Nuwe/Redigeer Menu

-- Voorlopige Weergawe
Menu Name: Type hier to create of redigeer 'n Menu Name.

Hierdie is die identifikasie naam used by Joomla to identifiseer hierdie menu metin the code ; it moet be unieke. Dit is recommended to gebruik no space metin hierdie name. For voorbeeld, in Joomla's core die Gebruiker Menu's Menu Name is gebruikermenu en die Main Menu se menunaam is mainmenu.

Stoor: Die Menu Name sal be updated of created in die Menu Bestuurder lys.

Wanneer creating 'n nuwe Menu Name:
Module Titel:
Tik hier 'n Module Naam.

Hierdie is die naam given na die nuwe mod_mainmenu module which is outomaties created when jy stoor hierdie menu. Dit is ook die Name of die menu soos dit sal verskyn on the bladsy if jy Kies to show it (this may be changed when redigeering die module). In die voorbeeld beskryf above these is Gebruiker Menu en Hoof Menu.

Stoor: 'n Nuwe mod_mainmenu module, met die Naam jy het entered above sal outomaties be created.
Parameters vir die module created is to be geredigeer deur die 'Modules Bestuurder [site]': Modules -> Werf Modules
