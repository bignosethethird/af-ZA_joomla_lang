Inhoud Items : New / Redigeer

--- Voorlopige Weergawe

Hierdie bladsy allows to create of redigeer 'n Inhoud Item.

Taakbalk Ikone

Voorskou: Opens 'n nuwe venster to preview die inhoud as jy would in die Frontend of jou werf.

Oplaai: Opens 'n nuwe venster wat jou toelaat upload 'n lêer of 'n prent van jou rekenaar na die images/stories folder.

Stoor: Stoor die inhoud en keer terug na die Bestuurder.

Toepas: Stoor die inhoud en behou die vertoonde inhoud om verder te redigeer.

Sluit: Kanselleer die aksie. Enige veranderings is verlore. Indien die item opened to redigeer, die Inhoud Item remains ingeteken.


Item Details

Titel: Type hier is die titel of die inhoud. Hierdie is die titel soos dit sal verskyn op die bladsy of table lysing hierdie inhoud.

Titel Alias: Hierdie field is not used by core. SEF scripts en dinamiese titel/meta scripts gebruik it en it may het 'n future use.

Seksie: Kies hier die Seksie jy want die item to belong to.

Kategorie: Kies hier die kategorie jy want die item to belong to. Die dropdown lys sal vertoon die beskikbaar Kategorië vir die Secton chosen above.

Intro Teks (required): Type hier die introductory portion of die inhoud. Hierdie is die part vertoonde in Blog Lists vir both Kategorië en Seksies. It kan ook be configured to be hidden when jy view die inhoud.

Main Teks (optioneel): Hierdie is die body van jou inhoud. Dit is only vertoon when viewing that inhoud in full view, en is not vertoon in Blog Lists.

Inhoud Inpropprogramme:
Inhoud Inpropprogramme is special tags jy place in jou inhoud that Joomla parses to perform 'n particular aksie of funksie. Hierdie Inpropprogramme sal be beskikbaar (if publiseered) even if jy Kies NO WYSIWYG redigeerder in die Globale Konfigurasie. They may be publiseered of geredigeer deur die Mambot Bestuurder (Inpropprogramme->Werf Inpropprogramme, Filter set to inhoud).

Die two verstek Inpropprogramme spesifiek used when redigeering inhoud are:
  • {mosimage} (mosimage Mambot): Used to tell Joomla where to place graphics in jou inhoud.
  • { mospagebreak} (mospaging Joomla): Instructs Joomla to create 'n bladsy break in jou inhoud. Jy kan break jou inhoud into several bladsys if jy like. Indien jy do, 'n table of inhoud sal vertoon to let jy navigate between die different bladsys created.

Knoppies to insert these codes may be publiseered of not deur (Inpropprogramme->Werf Inpropprogramme, Filter set to redigeerder.xtd). Jy mag ook manually type them into jou inhoud. Remember that in order vir jou images to show in die inhoud, jy moet insert the {mosimage} code snippet waar jy would like jou image to be placed in jou inhoud.

LET OP: Die images in die Intro Teks kan be hidden vir Blog Lists. Hierdie is 'n configurable setting vir these Menu Items. Also, hierdie is 'n global setting that sal affect jou entire web site as 'n verstek in Globale Konfigurasies. Indien jy is having trouble getting images to vertoon metin die Intro Teks en die problem keeps cropping up in Blog Lists, then make sure that die Parameters vir MOSImages is set to Show for that menu entry.

Formatting inhoud
Die formatting opsies interface depend op die Redigeerder jy is using.
Many redigeerders is beskikbaar as Open Source (deur http://ontwikkellaar.joomla.org) of commercial products.

Joomla comes met2 verstek redigeerders:
  • NO WYSIWYG: Hierdie redigeerder laat jou enter teks en/of HTML code directly.
  • TinyMCE: Hierdie redigeerder laat redigeer in WYSIWYG of HTML modus (See spesifieke Hulp).
LET OP: Dit is advised to publiseer only one redigeerder at 'n time in Mambot Bestuurder en to stoor ook dat choice in Globale Konfigurasie.


The Attributes Area vir inhoud consists of five tabs:
Publiseering, Images, Parameters, Meta Info en Skakel na Menu.

Publiseering
Publiseering Info

Show on Voorblad: Check die boksie if jy want hierdie inhoud to be assigned to the FrontPage Komponent.

Gepubliseer: Check die boksie to let die item show op jou werf.

Toegang Level: Kies wie sal be able to view hierdie inhoud. Public, registered or special.

Outeur Alias: Jy kan publiseer die outeur's naam vir 'n piece of inhoud. Jy kan Kies 'n Registered gebruiker as vertoonde deur die dropdown of type any naam there ('n outeur that does not het 'n account of inhoud created by many outeurs as 'n groep).

Change Creator: Hierdie allows jy to assign ownership of hierdie inhoud to ander registered gebruiker.

Ordering: Kies die order in which hierdie inhoud is lysed relative to others in hierdie Kategorie. Die choices beskikbaar in die dropdown sal show once changes to the item is saved of applied.

Override Created Date: Hierdie vertoon die created datum vir die inhoud. Jy kan wysig die datum if necessary.

Start Publiseering: Kies die datum when hierdie document is to be publiseered op die web site.

Finish Publiseering: Indien desired, jy kan set die datum when jou inhoud is to be expired.

Inhoud ID: Hierdie is die ID of die inhoud in die databasis.

State: Hierdie vertoon die current status of die document. Dit is not redigeerbaar.

Hits: Hierdie gives die aantal times hierdie inhoud was viewed. Die Reset Hits Value knoppie allows jy to reset die counter to zero.

Revised: Hierdie vertoon die aantal redigeers hierdie inhoud has undergone.

Created: Hierdie is die datum hierdie inhoud item was created.

By: Hierdie is die naam of die gebruiker that created die inhoud item.

Last Modified: Hierdie is die datum when hierdie inhoud item was last modified, if ever.

By: Hierdie is die naam of die gebruiker that last modified hierdie inhoud, if ever.


Images
MOSImage Beheer

Sub-folder: Kies hier die sub-vouer jy want to get die images van. Hierdie are in images/stories/.

Gallery Images: Hierdie dropdown lys die images beskikbaar in die spesifieke sub-vouer.
Highlight one of multiple image name(s) en kliek op die 'Add' button.
Die name(s) sal vertoon in die Inhoud Images dropdown.

Inhoud Images
Die 'up' en 'down' knoppies determine die order of images if there is meer than one on hierdie bladsy. Die images sal vertoon relative na die instances of {mosimage} tags in jou inhoud.

Redigeer die image selected
Kies 'n prent in 'Inhoud Images'.
Source: Vertoon die verstek naam of die image highlighted. Jy mag redigeer that name.
Image Align: Lets jy Kies between None, center, left of right.
Alt Teks: Lets jy Kies 'n Alt Teks vir die image.
Border: Type hier die aantal pixels jy want die border to be around die image.
Caption: Type hier 'n caption vir die image.
Caption Align: Lets jy Kies between None, center, left of right.
Width: Type hier 'n number in pixels to override die width of die image
Toepas: Kliek on hierdie knoppie to apply all die parameters defined vir hierdie image

Parameters
Parameter Beheer

Hierdie parameters control what jy see when jy view 'n Inhoud Item in full view.

Page Class Suffix: Allows jy to set customized styles vir hierdie bladsy. Die suffix jy type hier is outomaties added na die CSS classes vir hierdie bladsy, en sal het to be defined in jou templet's CSS stylesheet vir usage.

Back Knoppie: Beheers whether to show of hide 'n "Back" teks skakel that would verskyn op die bottom of die bladsy en return die gebruiker na die previously viewed bladsy. Die pull down laat select 'Use Globale' to gebruik die parameter as defined globally deur die Werf->Globale Konfigurasie, 'Hide' of 'Show'.

Page Titel: Teks to vertoon at die top of die bladsy.

Show/Hide Page Titel: Hides of vertoon die Page Titel teks (defined in die field beskryf above).

Seksie Name: Hides of vertoon die naam of die seksie to which elke item belongs. Indien hierdie option is turned on, elke item's seksie naam sal be vertoon under die item's titel op die home bladsy.

Seksie Name Linkable: Ontspers jy to make elke item's seksie naam linkable. Applies only when die Seksie Name option (beskryf above) is set to 'Show.'

Kategorie Name: Hides of vertoon die naam of die kategorie to which elke item belongs. Indien hierdie option is turned on, elke item's kategorie naam sal be vertoon under die item's titel op die home bladsy.

Kategorie Name Linkable: Ontspers jy to make elke item's kategorie naam linkable. Applies only when die Kategorie Name option (beskryf above) is set to 'Show.'

For elke of die volgende eight items lysed below, jy kan gebruik die option selected in die Globale Konfigurasie settings (Werf -> Globale Konfigurasie) by selecting 'Use Globale' of jy kan override die Globale Konfigurasie setting by selecting 'Show' of 'Hide.' Overriding die settings hier vir items sal not affect die Globale Konfigurasie settings en sal not affect die settings op die rest of die site.

Linked Titels: Ontspers jy to make item titels linked. Die skakel sal go na die item bladsy.

Item Rating: Hides of vertoon ratings given to home bladsy items by jou werf's gebruikers.

Outeur Names: Hides of vertoon die name of outeurs of items.

Created Date en Tyd: Hides of vertoon die datum en time that items were created.

Modified Date en Tyd: Hides of vertoon die datum en time that items were last modified.

PDF Ikoon: Hides of vertoon die ikoon en skakel na die PDF weergawe of items.

Afdruk Ikoon: Hides of vertoon die ikoon en skakel na die printer-friendly format of items.

E-pos Ikoon: Hides of vertoon die ikoon en skakel na die bladsy that enables gebruikers to email items to others.

Key Reference: 'n teks key that die item may be referenced by (like 'n hulp reference).

Docbook Type: Experimental, do not gebruik in production work. Associates 'n type of DocBook document to hierdie item.

Meta Info
Meta Data

Beskrywing: Hierdie inligting is inserted in die headers van jou inhoud. Dit is commonly used by search engines.

Keywords: Hierdie sleutelwoorde is related na die header inligting provided vir search engines, maar they ook provide ander important funksie. Hierdie sleutelwoorde drive die What's Verwandte Module that provides automatic links to other inhoud items that shares 'n similar sleutelwoord.

Add Sect/Cat/Titel: Klieking on hierdie knoppie is 'n convenient way to insert die inhoud's inligting inna die sleutelwoorde field, including die inhoud's titel, seksie name, en kategorie name.

Skakel na Menu

Hierdie sal create 'n 'Link - Inhoud Item' in die menu jy select.

Kies 'n Menu: Highlight hier once die menus defined vir jou werf.

Menu Item Name: Die naam of die Menu soos dit sal verskyn op jou werf.

Skakel na Menu: Kliek on hierdie knoppie to create die Menu Item.


Bestaande Menu Items

Hierdie vertoon 'n lys of all 'Link - Inhoud Item' Menu Items referencing hierdie inhoud.
