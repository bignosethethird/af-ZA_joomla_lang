Beheer Paneel

Die Beheer Paneel is die eerste bladsy jy see when jy log inna die Administrateurs area. At anytime, jy kan return to it by kliek die "Home" menu item in die top menu bar. From die Beheer Paneel is 9 shortcut ikone to easy access die frequently used parts of die Administrateur site.


Beheer Paneel Ikone:
Menu Bestuurder - Kliek hierdie ikoon to access die Menu Bestuurder. You sal be able to redigeer any van jou menus.

Front Page Bestuurder - Kliek hierdie ikoon to access die Front Page Bestuurder. You sal be able to redigeer which Inhoud items verskyn op jou Voorblad.

Statiese Inhoud Bestuurder - Kliek hierdie ikoon to access die Statiese Inhoud Bestuurder. You sal be able to redigeer these type of items.

Seksie Bestuurder - Kliek hierdie ikoon to access die Seksie Bestuurder. You sal be able to redigeer jou Seksies.

Kategorie Bestuurder -Kliek hierdie ikoon to access die Kategorie Bestuurder. You sal be able to redigeer jou Kategorië.

All Inhoud Items - Kliek hierdie ikoon to access die All Inhoud Items. All Inhoud Items lys all van jou Inhoud Items. You sal be able to redigeer any van jou Inhoud Items.

Vuillis Bestuurder - Kliek hierdie ikoon to access die Vuillis manager. You sal be able to skrap permanently of to restore any items in die Vuillis Bestuurder.

Globale Konfigurasie - Kliek hierdie ikoon to access die Globale configuration. You sal be able to redigeer any van jou webwerf configurations.

Hulp - Kliek hierdie ikoon to view die Index of any of die Hulp Skerms.

On die right side of die skerm there is four tabs:

Logged - Hierdie vertoon 'n lys of gebruikers currently logged on (both in die Admin and die Frontend seksies). Kliek 'n gebruikers naam to go na die Gebruiker Redigeer bladsy vir that gebruiker.

Komponente - Kliek hierdie shortcut tab to access any of die komponente en component links that jy wish to redigeer.

Populêr - Kliek hierdie tab to access die skerm which lys in order, die most popular Inhoud items op jou werf. Hierdie includes die datum die Inhoud item was created en die aantal hits. Kliek die Inhoud Item naam to redigeer that item.

Latest Inhoud - Kliek hierdie tab to access die skerm which lys die most recently added Inhoud items. Hierdie includes die creation datum en die outeur. Kliek die Inhoud item naam to redigeer that item.

Menu Stats - Kliek hierdie tab to access die skerm which lys all menus en die number of item links in elke menu. Die aantal links is die total created, not die total publiseered. Kliek die naam of 'n menu to access die skerm which lys the items vir that menu.
