Globale Konfigurasie

Globale Konfigurasie beheer Joomla se operasionele instellings. Veranderinge op hierdie skerm sal die configuration.php lêer opdateer. Hierdie lêer moet skryfbaar wees sodat jy jou konfigurasie kan redigeer.

Aan die regte kant van die skryf-status van die configuration.php is 'n merkboksie genoem "Maak onskryfbaar nadit dit gestoor is". As jy die boksie tik, sal die lêertoegangsregte van configuration.php na onskryfbaar gestel word. Wanner jou werf gekonfigureer is, is dit op sekuriteitsgronde altyd 'n goeie idee om die configuration.php-lêer onskryfbaar te maak.


Werf Ootjie:

Werf van lyn af - Hiermee kan jy vinnig en effektief jou werf se voorkant aan en af skakel. Gebruik hierdie opsie terwyl jy modules of data opgradeer en nie wil hê dat jou besoekers die werf sien todat jy nie klaar is nie. Onthou om dit "Werf van lyn af" opsie weer na "Nee" te herstel wanner jy klaar is.

Van lyn af Boodskap - Wanner jou Werf van lyn af skakellaar op die Ja-posisie is, sal hierdie verstek boodskap op jou werf se voorkant vertoon word wat jou besoekers dan sal sien: "Hierdie webwerf is van lyn af vir instandhouding. Kom kuier binnekort weer, asb!."
Jy kan hierdie teks na enige ander boodskap verander.

Stelsel Fout Boodskap - Hierdie message sal vertoon if die initial databasis connection fails. Jy kan type anything jy like in hierdie boksie. We suggest something friendly en apologetic, stating that jou werf sal be beskikbaar again as soon as possible. Jy kan gebruik standard HTML code in hierdie boksie ook, vir instance, <br /> to return 'n line. You might ook consider adding 'n e-posadres, so dat jou besoekers kan kontak jou Stelsel Administrateur of die Webmaster.

Werf Name - Hierdie option allows jy to set die inhoud of die titel bar vir jou site. It kan ook be vertoonde in jou templet by inserting die volgende code statement into jou templet lêer [Joomla_root/templette/templet name/index.php] waar jy wish jou Werf Name to verskyn:

%%<?php echo $_CONFIG->SITENAME; ?>%%

Show Unouteurised Skakels - With hierdie option turned on, all besoekers sal see inhoud to which they would not otherwise het access, (that is, inhoud given 'n Toegang Level of 'Registered' of 'Special'). With hierdie option off they sal only see inhoud metan Toegang Level of 'Public'. Die toegangsvlak vir Inhoud is selected under die Publiseering tab in die Redigeer Inhoud screen.

Allow Gebruiker Registration - Hierdie option allows jy to globally set whether besoekers to jou werf kan register vir 'n gebruiker account. Indien die "Yes" radio button is chosen, die Login Form sal include die prompt 'No account yet? Create one'?. Indien die "No" radio knoppie is chosen, they sal not be presented metan option to register.

Use New Account Activation - Indien die "Yes" radio knoppie is chosen, 'n activation skakel sal be sent to gebruiker's e-pos address. Gebruikers kan not login until he has clicked op die activation link. Indien die "No" radio knoppie is chosen, Gebruikers sal be able to login right away sonder activation.

E-pos New Gebruiker Password - With hierdie option set to 'Ja', die new gebruiker sal receive 'n email meta wagwoord generated by Joomla. Indien it is set to 'Nee', new Gebruikers sal select their own wagwoord.

Require Unique E-pos - Kies whether jy want to prevent registration metduplicate e-pos addresses. In combination metgebruiker activation hierdie option prevents gebruikers to sign up several times using dieselfde e-pos address.

Debug Werf - Hierdie option allows jy to view die error messages that is created by jou werf's php installation. It kan be 'n handy tool vir debugging if jy are having problems en vir reporting errors to jou Stelsel Administrateur of Web host. Die levels of error reporting kan be set under die Globale Konfigurasie>Server tab, en is as follows:
Stelsel Verstek, None, Eenvoudige en Maximum.

WYSIWYG Redigeerder - Kies which of die installeer Teks Redigeerders jy want to gebruik to redigeer Inhoud Items.

List Length - Kies 'n verstek length vir lys in die Administrateur (Back-end). Please LET OP that if jy het previously visited 'n bladsy met'n lys, changing this setting in Globale Konfigurasie sal not be immediately visible. However, die setting jy specify sal be honoured as 'n verstek upop jou next login.

Favourites Werf Icon - Hierdie sets die ikoon that verskyns in some browsers langs die naam en/of address of jou werf. By verstek hierdie is die Joomla flower. Indien jy want ander ikoon to show up, jy moet upload ander icon to vervang die Joomla flower. Dan type in die address of die image in this field.


Locale Tab:
Taal - Kies die verstek taal used op jou werf. Indien it is ander taal other than English, jy moet install die taal l lêersecirc; lêerseuml;rs eerste using the Installeerder menu in die Back-end.

Tyd Offset - Hierdie option sets die time offset in hours between jou server location en GMT (Greenwich Mean Tyd - UK). Kies die offset time van die dropdown menu.

Country Locale - Type die location van jou webwerf. For instance en_GB would be die setting vir die UK.


Inhoud Oortjie:
Linked Titels - Kies 'Ja', if jy want die titel of 'n inhoud item to be hyper-linked. Kies 'Nee', en 'n 'Read On...' skakel sal verskyn below die introduction text.

Read More Link - Kies "Show" to vertoon die teks "Read more" at die end van jou "Intro Teks" van jou inhoud items.

Item Rating / Voting - Hierdie option allows die rating of inhoud items. Kies 'Ja', en 'n 'Star Bar' sal be vertoon on elke inhoud item, as well as 'n form vir submitting jou rating. LET OP: The rating form is not vertoon vir articles in 'Blog' lys, en die Star Bar is only vertoon vir those items metIntro text.

Outeur Naam - Kies die radio knoppie "Vertoon", om die Outeur se naam te vertoon wanneer 'n inhoudsitem vertoon word.

Created Date en Tyd - Kies die radio knoppie "Show", to allow die creation datum en time to be vertoon when 'n inhoud item is vertoonde.

Modified Date en Tyd - Kies die radio knoppie "Show", to allow the last modified datum en time to be vertoon when 'n inhoud item is vertoonde.

Hits - Kies die radio knoppie "Show", to vertoon die aantal hits 'n article/inhoud has had.

PDF Icon - Kies die radio knoppie "Show", to show die PDF maker icon on Inhoud Items while viewing in die Front-end.
LET OP: Hierdie sal not vertoon at all if jou media lêergids is not writable.

Afdruk Icon - Kies die radio knoppie "Show", to show die Afdruker Friendly Version ikoon on Inhoud Items while viewing in die Front-end.

E-pos Icon - Kies die radio knoppie "Show", to show die E-pos Page to Friend ikoon on Inhoud Items while viewing in die Front-end.

Ikone - Kies die radio knoppie "Show", when jy want die hidden links to E-pos/Afdruk/PDF to be ikone rather than text.

Tabel of Inhouds on multi-page items - Kies die radio knoppie "Show", to show dieTabel of Inhoud navigation index boxes vir multi-page Inhoud Items. To create bladsy breaks in jou Inhoud Item, jy moet insert 'n bladsy break Mambot, {mospagebreak} using die redigeerder where ever jy want die bladsy breakes to occur.

Terug Knoppie - Kies die radio knoppie "Show", to allow jy to show navigational back knoppies on elke bladsy to make it easier vir gebruikers to navigate jou werf.

Inhoudsitem Navigasie - Kies die radio knoppie "Show", to publiseer 'Next' en 'Previous' links when viewing Inhoud Items in 'n kategorie or to wysig bladsys metin dieselfde article.


Databasis Oortjie:
Waarskuwing: Hierdie is very important settings. If jou Joomla webwerf has been installeer op jou behalf do not alter these settings.

Hostname - Type 'localhost'. as hierdie is die typical Host name

My SQL Gebruikernaam - Redigeer, if needed, die gebruikername to access jou Joomla databasis.

My SQL Password - Redigeer, if needed, die wagwoord to access jou Joomla databasis.

My SQL Database - Redigeer, if needed, die naam of die databasis jou Joomla installation uses.

My SQL Database Voorvoegsel - Wysig, indien nodig, die voorvoegsel wat vir tabelle in jou Joomla databasis gebruik word. Gewoonlik is dit "mos_".
Waarskuwing: Moet die nie verander, behalwe as jy reeds 'n databasis het wat 'n ander voorvoegsel in sy tabelle gebruik!


Server Tab:
Hierdie items is gedurende die installeering gedefineer en is nie hier konfigureerbaar nie.

Absolute Gidspad - Hierdie is die lêergids wat Joomla! op die bediener gebruik sodat dit t.o.v. die omgewing kan oriênteer.

Lewendige Werf - Hierdie is die absolute bronadres vir jou Joomla!-installasie.

Geheime Woord - Hierdie is die unieke word vir elke Joomla installasie. Dit word vir sekuriteitsfunksies gebruik.

GZIP Page Compression - Kies die radio knoppie "Yes", to allow jou web server to send compressed data, if supported by die server, to the client web browser. Hierdie could significantly decrease die load time of jou werf vir jou besoekers.

Login Session Lifetime - Type die aantal sekondes die Administrasie session should be allowed to stay live before automatic logout occurs. After automatic logout, die Administrateur sal be required to login again using die correct gebruikername en wagwoord.

LET OP: Indien jy is redigeering 'n Inhoud Item, maar het not saved jou work, metin die aantal sekondes choosen, jy sal be logged out outomaties .

Fout Rapporteering - Indien hierdie opsie ontsper is under die Werf tab, then using die aftrek boksie, select die level of reporting required.
Die opsies is: Stelsel Verstek, Geen, Eenvoudig en Maksimum.

  • Stelsel Verstek - Hierdie turns Error reporting na die system verstek according to php.ini op die server.
  • None - Hierdie turns off error reporting.
  • Eenvoudige - Hierdie turns error reporting to simple opsies using die volgende opsies: error_reporting (E_ERROR|E_Waarskuwing|E_PARSE)
  • Maximum - Hierdie turns error reporting na die Maximum level using the volgende opsies: error_reporting (E_ALL)

Lêer Creation - By verstek, l lêersecirc; lêerseuml;rs created by die installation of komponente, modules en mambotte inherit die server verstek permissions. By choosing "CHMOD new l lêersecirc; lêerseuml;rs", jy kan over-ride die server permissions.

Directory Creation - As metl lêersecirc; lêerseuml;rs, new directories sal by verstek inherit the server verstek permissions. Kies "CHMOD new directories" to over- ride die server permissions.


Metadata Tab:
Globale Werf Meta Beskrywing - Type 'n beskrywing van jou web site used to assist Soektog Engines. Jy mag want to limit jou beskrywing to maybe twenty words of so, depending op die Soektog Engine jy is targeting. Make it concise en pertinent to jou werf's inhouds. Jy kan include some van jou sleutelwoorde en key frases hier too. As some search engines read meer than 20 words jy kan add one of two sentences. Make sure that die most important part van jou beskrywing is in die eerste 20 words.

Globale Werf Meta Keywords - Die optimal aantal characters vir 'n sleutelwoord beskrywing is said to be between 300 en 500. Do not repeat words too often en òf separate jou sleutelwoorde meta comma of 'n space, of groep two of three words together. 'Joomla Open Source' would be 'n voorbeeld of this.

Show Titel Meta Tag - Shows die Titel meta tag in die source code when viewing inhoud.

Show Outeur Meta Tag - Shows die Outeurs meta tag in die source code when viewing inhoud.

LET OP: The Beskrywing en Keyword Meta tags sal het 'n bearing on how Soektog Engines sal index jou werf. Party Soektog Engines completely ignore Metadata altogether. You should do some research on Soektog Engine Optimization if high rankings en Soektog Engine traffic is important to jy.


Mail Tab:

Mailer - Kies which mailer jy wish to use: PHP mail funksie, Sendmail of SMTP Server.

Mail From - Kies which e-posadres sal show as die sender.

From Name - Kies what naam sal show as die sender.

Sendmail Path - Hierdie is die lêergids of die Mail server

SMTP Auth - Kies "Yes" if SMTP Authentication is required by jou host.

SMTP Gebruiker - Indien authentication is required en set above, type jou gebruikername.

SMTP Pass - Indien authentication is required en set above, type jou wagwoord.

SMTP Host - Hierdie is usually set as localhost, maar jy may need to check hierdie metyour host.


Cache Tab:
Caching - Kies "Yes" to allow vir caching of certain elements metin Joomla. Hierdie sal generally improve die performance of jou werf.

Cache Folder - Hierdie is die folder of lêergids to store die cache l lêersecirc; lêerseuml;rs. Hierdie moet be writable vir hierdie caching to be successfully implemented. Kontak jou Web Master if jy het any difficulties.

Cache Tyd - Redigeer die maximum length of time, in sekondes, vir 'n cache lêer to be stored before it is refreshed.


Statistieke Tab:

Statistieke - Kies 'Ja', to allow die browser en visitor domain statistics to be collected op die basis of settings in die Inhoud Tab. Indien hierdie is set to 'Nee', die browser en visitor statistics sal not be collected. Indien jou site experiences 'n slight 'delay' die eerste time 'n gebruiker visits die site jy should set hierdie to 'Nee'.

Log Inhoud Hits by Date - Kies "Yes" to enable logging of die hits on Inhoud Items, on 'n daily basis. There is currently no in-built analysis routines, to complement hierdie logging feature.
Waarskuwing: This sal tend to collect masses of data over time. Be sure that jou server space has die space to to store this inligting.

Staaf Soektog Stringe - Kies "Yes" to enable logging of die search text, entered by 'n gebruiker. Such logging facilitates die 'tuning' van jou web site, giving jy 'n insight into how jou besoekers is trying to find die inligting they seek. Analysis reporting is provided under Werf->Statistieke->Soektog Teks.


SEO Tab:
Soektog Engine Friendly URLs: Hierdie changes die way site links is presented en optimises them so dat search engines kan access meer of jou werf. Hierdie option is ONLY beskikbaar to sites hosted op Apache bedieners. Kies 'Ja' to enable Joomla to output friendly URLs rather than normal databasis generated URLs. Indien jy gebruik hierdie option jy moet renaam jou htaccess.txt file to .htaccess.

Dinamiese Bladsy Titels - Kies "Ja" to set die bladsy titels (that verskyn at die top of 'n viewers browser) to be dieselfde as die bladsy titel they are viewing. Again hierdie helps search engines.

LET OP: Jy moet jou veranderinge 'Stoor' wanneer jy klaar is, met die 'Stoor'-ikoon in die taakbalk.
