Inpropprogramme: New / Redigeer

The Inpropprogramme: New/Redigeer bladsy laat jou redigeer die Werf Inpropprogramme.
Detailed funksies vir elke verstek Mambot is provided below.

Taakbalk Ikone:
Stoor: Kliek to stoor die parameters entered en return na die Mambot Bestuurder.

Toepas: Kliek to stoor die parameters entered AND keep dieselfde venster open vir redigeering.

Sluit: Kliek to cancel any verandering just made en to sluit die venster. Enige verandering is lost. Die Mambot remains ingeteken.


Mambot Details:

Name: Type die naam of die Mambot as it sal verskyn op die Werf Inpropprogramme lysing. Hierdie has no impact op jou werf.

Folder: Hierdie vertoon into which kategorie die Mambot is placed.

Mambot lêer: Hierdie is die naam of die Mambot lêer as it verskyns in die folder.

Order: Kies die order of die Mambot as lysed in die Folder when filtering by kategorie.

Toegang Level: Kies wie sal be able to view this inhoud. Public, Registered of Special.

Gepubliseer: Check die boksie to let die Mambot show op jou werf of provide die funksie it is designed to do.

Beskrywing: Hierdie is 'n kort beskrywing on the Mambot's funksie.


Parameters:
Indien beskikbaar, they sal be vertoonde hier.


Werf Inpropprogramme provided in 'n verstek installation of Joomla:
MOS Image:
Usage: { mosimage }

Hierdie mambot replaces tag in inhoud metspesifiseerde images. Jy kan insert as many of these tags as needed in jou articles. To do this while creating of redigeering 'n inhoud item, select die images op die "Image" tab under "MOSImage Beheer". Die replacement of die tags sal follow die order of die images in die "Inhoud Images" boksie.
LET OP: The tag beskryf above should het no space after "{" en before "}". It has been vertoonde metspaces hier not to implement die funksie in hierdie screen.


GeSHi:

Hierdie vertoon formatted code in inhoud items gebaseer op die GeSHi highlighting engine.

Hierdie is to be considered as 'n enhanced weergawe of mosCode, allowing die rendering of many different programming taals metcolor-highlighted code by specifying die to-be-used taal.

Usage: Use die "pre" tag en add die programming taal jy want to gebruik vir highlighting purposes as 'n parameter. Example:

< pre lang="xml" >jou code hier ... < /pre >

Die mambot weergawe of die GeSHi highlight code comes met die volgende highlighters vir die most common renders installeer, ready to be used:

  • CSS => gebruik lang="css"
  • Html4 Strict => gebruik lang="html4strict"
  • JavaScript => gebruik lang="javascript"
  • PHP => gebruik lang="php"
  • PHP Brief => gebruik lang="php-brief"
  • SQL => gebruik lang="sql"
  • XML => gebruik lang="xml"

Should jy need additonal taals, jy may download die full paket van the ontwikkellaar site: http://qbnz.com/highlighter/ which includes over 30 taals. You should then upload die required "newTaal.php" van die paket into jou "\mambotte\inhoud\geshi\geshi" folder.

LET OP: As of 4.5.2.1, hierdie bot seems to het 'n incompatibility met die E-pos cloaking mambot.


Legacy Mambot Includer:

Hierdie is ondersteun vir outdated Joomla 4.5 mambotte.
Hierdie Mambot is ontpubliseered by verstek; should jy experience disfunksies meta particular, older mambot, try to publiseer it.


Code Ondersteuning:

Hierdie vertoon formatted code in inhoud items. No syntax highlighting. Code presented in Courier.

Usage: { moscode }... { /moscode }

Example:
{ moscode }<body onload="winBestuur(850,700,true);"><'n name="top"></'n><div><div id="wrapper"></div></div>{ /moscode }
LET OP: Die tag beskryf above should het no space after "{" en before "}". It has been vertoonde metspaces hier not to implement die funksie in hierdie screen.


MOS Rating:

Hierdie sal add die Voting funksienaliteit to Inhoud Items.

LET OP: For hierdie mambot's output to be visible, jy moet set "Item Rating/Voting:" to "Show"
deur Werf- > Globale Konfigurasie- > Inhoud.

Gebruikers sal then be presented meta rating system van 1 to 5 at die top of 'n full inhoud item. They sal be able to cast their vote (once).


E-pos Cloaking:

Hierdie hides all e-poss in inhoud van spambots using javascript. It does not funksie vir Link - URL Menu Items. By redigeering die mambot, jy may select between 2 settings under Parameters -> Mode:
  • Nonlinkable Teks sal vertoon 'n email typed by die gebruiker as simple teks (not linked)
  • As linkable mailto address sal vertoon die e-posadres as 'n skakel to send 'n email, using die spesifiseerde address.
As of 4.5.2.1 these is die ways die address should be written:
1. In plain text: mail@mail.com
2. As 'n mailto: <'n href="mailto:mail@mail.com">mail@mail.com</'n>

Otherwise die cloaking prevents die address van being decrypted when calling the mailer.

As of 4.5.3 CVS, die mambot sal accept:
<'n href="mailto:mail@mail.com">Party text</'n>


Load Module Posisies:

Hierdie loads module posisies metin Inhoud.
Usage: {mosloadposisie gebruiker1} where 'gebruiker1' is one of die posisies defined vir jou werf deur Werf->Templette->Modules Posisies.

Hierdie means jy may load any module assigned to hierdie posisie at hierdie spesifieke place into jou Inhoud sonder defining die posisie inna die Templet index.php lêer.

Parameters laat jou Kies 'n style of HTML code to insert die module:
  • Wrapped byTabel - Column
  • Wrapped byTabel - Horizontal
  • Wrapped by Divs
  • No wrapping, raw output

MOS Pagination:

Hierdie sal add aTabel of Inhouds funksienaliteit to 'n paginated item.
Usage: { mospagebreak } of { mospagebreak titel=The bladsy titel } of { mospagebreak heading=The eerste bladsy } of { mospagebreak titel=The bladsy titel&heading=The eerste bladsy } of { mospagebreak heading=The eerste bladsy&titel=The bladsy titel }

'n Nuwe 'inhoudtoc' CSS class word gebruik to vertoon thisTabel of Inhouds.

LET OP: Die tags beskryf above should het no space after "{" en before "}". They het been vertoonde metspaces hier not to implement die funksie in hierdie screen.

Parameters:
Werf Titel: Hierdie laat jou add of not die mambot titel en heading attributes na die Werf Titel Tag.


No WYSIWYG Redigeerder:
Hierdie loads 'n basic teks entry field wherever jy may redigeer inhoud.


TinyMCE Redigeerder:
Hierdie loads hierdie verstek WYSIWYG Redigeerder wherever jy may redigeer inhoud.


MOS Image Redigeerder Knoppie:
Hierdie provides die knoppie to insert die MOS Image tag into inhoud redigeer boksie.


MOS Pagebreak Redigeerder Knoppie:
Hierdie provides die knoppie to insert die { mospagebreak } tag into inhoud redigeer boksie.
