Sindikeering Instellings

-- Voorlopige Weergawe

To syndicate jou werf is to make die Inhoud Items van jou webwerf assigned na die FrontPage Komponent beskikbaar vir Sindikeering via 'n distributable lêer.

Cache: Ja of Nee? Kies die 'Ja' radio button to create and vertoon 'n nuwe Nuus Afvoer dependant op die value set in die Cache Tyd field. Kies die 'Nee' radio knoppie to vertoon 'n nuwe Nuus Afvoer elke time 'n item is added na die FrontPage Komponent and elke time die Nuus Afvoer is accessed by ander server.

Waarskuwing: Setting Cache to 'Nee' sal cause jou Nuus Afvoer to be checked elke keer dat iemand visits any site waar jyr Nuus Afvoer is syndicated. Hierdie kan potentially cause excessive bandwidth usage as well as extreme draw op jou server resources.

Cache Tyd: Hierdie is die aantal sekondes before jou site's databasis is checked vir updates. die default in Joomla is 3600 sekondes, or 60 minutes. After hierdie time has elapsed, die News Feed sal refresh itself van jou databasis thus vertooning any new items jy het publiseered deur die FrontPage Komponent to jou Nuus Afvoer.

#Items: Type die aantal items jy would like to syndicate and vertoon in jou Nuus Afvoer.

Titel: Type die titel jy wish to vertoon above jou articles metin jou syndicated Nuus Afvoer when it is vertoonde on those sites and pages which syndicate jou Nuus Afvoer. Die titel sal verskyn as 'n skakel to jou werf.

Beskrywing: Type 'n brief statement to describe jou Nuus Afvoer. Indien none is entered, it sal verstek to "Joomla site Sindikeering".

Image: Kies 'n prent van die aftreklys menu to be included in jou Nuus Afvoer. Indien jy do not het 'n prent, jy kan upload one via die Media Bestuurder.

Image Alt: Type die text vir die 'alt'? tag to compliment die image jy've selected vir inclusion in die Nuus Afvoer via the Image aftreklys menu.

Limit Teks: Ja of Nee? Kies die 'Ja' radio button if jy wish to limit die Text Length field. Kies die 'Nee' radio knoppie to force die value in die Teks Length field to be ignored, thus vertooning die entire inhouds of elke item.

Text Length: Type die aantal words to vertoon per Nuus Afvoer item.
LET OP: die Limit Teks radio knoppie moet be set to 'YES' to activate hierdie funksie.

Order: Order that die items '?as lysed in die FrontPage Bestuurder'? sal be vertoonde.

  • 'Verstek': Arranges items according to the order set in die FrontPage Bestuurder.
  • 'FrontPage Ordering': Dieselfde as above.
  • 'Oldest eerste': Arranges items in chronological order, met die oldest items eerste.
  • 'Most recent eerste': Arranges items in chronological order, met die newest items eerste.
  • 'Titel Alphabetical': Arranges items by titel in alphabetical order.
  • 'Titel Reverse-Alphabetical': Arranges items in reverse-alphabetical order.
  • 'Outeur Alphabetical': Arranges items by outeur in alphabetical order.
  • 'Outeur Reverse-Alphabetical': Arranges items by outeur in reverse alphabetical order.
  • 'Most Hits': Arranges items according to the aantal hits, met die items met die most hits eerste.
  • 'Least Hits': Arranges items according to the aantal hits, met die items met die least hits eerste.

Live Bookmarks: Activate ondersteun vir Firefox Live Bookmark funksienaliteit.
Bookmark Lêer: Special lêer name. Indien none is entered, sal gebruik die verstek lêer.
