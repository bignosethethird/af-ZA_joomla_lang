Modules: New

Die Gebruiker module is 'n custom module created when kliek the 'New' ikoon in die Module Bestuurder.
Jy mag gebruik hierdie module to vertoon Inhoud (this inhoud is not related to Seksies, Kategorië of Statiese Inhoud Items) of 'n RSS Feed.

Type: mod_gebruiker
Details:
Titel: Type of redigeer die titel of die module to vertoon op die Front-end of Back-end of jou werf.

Show titel: Kies if jy want die Titel to vertoon of not on op die Front-end or Back-end of jou werf.

Posisie: Kies which posisie that jy want hierdie module to vertoon in. Hierdie is determined die posisies beskikbaar in jou templette.

Module orde: Kies die order hierdie module sal verskyn among modules assigned to dieselfde posisie in jou templette.

Toegangsvlak: Kies wie kan view hierdie module. Public, Registered of Special.

Gepubliseer: Kies if die module inhouds sal show op die Front-end van jou Web site (if die chosen Posisie is beskikbaar in die templet).

ID: Hierdie is die ID number vir die module item. Hierdie is auto-allocated by die Joomla system when die module is saved.

Pages / Items: Kies which bladsys die module should be vertoonde on.


Parameters:
Module Class Suffix: Add 'n suffix if needed to be applied na die CSS class of the module to allow individual module styling.

Ontsper Cache: Kies to cache of not die inhoud of hierdie module.


Hierdie inligting does not concern RSS Feeds:
Custom Output: Hier jy may enter any text of code jy like that jy wish to vertoon it's output in ths module anywhere op die bladsys of jou werf.
LET OP: Hierdie inhoud may not be accessed deur 'n Menu Item.


Hierdie inligting concerns RSS Feeds:

RSS URL: Enter hier die URL of die RSSfeed jy want to vertoon op jou site.

Feed beskrywing: Kies to show of not die beskrywing teks vir die whole Feed.

Feed Image: Kies to show of not die image associated met die whole feed.

Items: Enter die aantal RSS Items to vertoon. Indien hierdie number is higher than the number fixed by die Syndicating site, die latter one sal be used.

Item beskrywing: Type die beskrywing of intro teks vir individual News Items.

Word Count: Kies die die maximum aantal visible words beskikbaar van die beskrywing of die New Item. 0 means all teks sal be vertoonde.
